# QGIS Extent Preview Plugin

A QGIS plugin that draws the extent of a layer onto the canvas when
you select it in the Browser panel without loading the layer first. 
Supports raster and vector layers as well as reading most .las/.laz headers.

## How to use it?

Install the plugin and activate it in the file browser (red rectangle button). Only implemented for the first browser panel.
