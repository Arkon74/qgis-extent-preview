# QGIS Extent Preview Plugin

Preview the extents of files in the map canvas. Supports raster and vector layers as well as reading most .las/.laz headers.

## How to use it?

Install the plugin and activate it in the file browser (red rectangle button). Only implemented for the first browser panel.
